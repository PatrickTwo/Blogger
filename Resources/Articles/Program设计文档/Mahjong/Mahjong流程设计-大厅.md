# 麻将游戏大厅流程设计文档

## 一、PageUI: LobbyPageUI

### 1.1 界面布局
1. **左上角区域**:
   - 开始游戏按钮(StartGameBtn)
   - 游戏玩法设置按钮(PlaySettingBtn)

2. **右上角区域**:
   - 游戏设置按钮(GameSettingBtn)
   - 加入房间按钮(JoinRoomBtn)
   - 房间号显示(RoomID)

3. **中央区域**:
   - 玩家列表（四个玩家位置）

4. **右下角区域**:
   - 麦克风开关(MicTog)
   - 扬声器开关(SpeakerTog)

5. **左下角区域**:
   - 聊天框(ChatBox)

### 1.2 核心功能逻辑

#### 开始游戏按钮(StartGameBtn)
**前置验证条件**:
1. 验证玩家人数是否足够（必须四名玩家才可开始）
   - 不足时弹框提醒："玩家人数不足，需要4名玩家才能开始游戏"
2. 验证所有玩家是否都已准备就绪
   - 未准备时弹框提醒："还有玩家未准备，请等待所有玩家准备"

**成功条件**:
- 所有玩家准备就绪且人数足够时，进入选庄状态(GameState.SelectBanker)

#### 玩家列表管理
**玩家位置布局**:
- 四个横向分布的固定占位框
- 每个占位框包含以下功能按钮：
  - 邀请按钮：邀请好友加入
  - 添加人机按钮：添加AI玩家

**玩家信息显示**:
- 玩家头像（圆形显示）
- 玩家昵称（显示在头像下方）
- 准备状态指示器（绿色表示已准备，灰色表示未准备）

**玩家操作菜单**:
- 点击玩家头像展开操作面板
- 可执行操作：
  - 查看玩家信息
  - 踢出房间（房主权限）

#### 音频控制逻辑
**麦克风(MicTog)**:
- 默认状态：关闭
- 开启条件：扬声器必须已开启
- 状态切换：点击切换开/关状态

**扬声器(SpeakerTog)**:
- 默认状态：开启
- 状态切换：点击切换开/关状态

**音频权限控制**:
- 麦克风开启时自动请求音频权限
- 权限被拒绝时显示提示信息

## 二、PanelUI: 面板系统设计

### 2.1 面板列表
1. **进房面板** - JoinRoomPanelUI
2. **对局玩法设置面板** - PlaySettingPanelUI
3. **游戏设置面板** - GameSettingPanelUI
4. **聊天框面板** - ChatBoxPanelUI
5. **玩家操作面板** - PlayerOperationPanelUI
6. **玩家信息面板** - PlayerInfoPanelUI
7. **提示框面板** - PromptPanelUI

### 2.2 面板详细设计

#### 2.2.1 进房面板 JoinRoomPanelUI
**界面元素**:
- 房间号输入框（6位数字，自动格式化）
- 加入房间按钮

**交互逻辑**:
1. 房间号验证：
   - 必须是6位数字
   - 实时验证格式正确性
2. 加入房间流程：
   - 验证房间是否存在
   - 验证房间是否已满
3. 错误处理：
   - 房间不存在："房间不存在，请检查房间号"
   - 房间已满："房间已满，请选择其他房间"

#### 2.2.2 对局设置面板 PlaySettingPanelUI
**三级设置结构**:

**玩法设置**:
- 麻将类型选择：
  - 川麻、大众麻将、红中麻将、广东麻将等
- 具体规则配置：
  - 是否可以吃牌
  - 是否可以碰牌
  - 是否可以杠牌
  - 特殊规则（如血流、血战等）

**游戏规则设置**:
- 出牌回合时长：10秒/15秒/30秒（可自定义）

**计分设置**:
- 底分设置
- 玩家初始分数

**局数设置**:
- 游戏局数

#### 2.2.3 游戏设置面板 GameSettingPanelUI
**音效设置**:
- 主音量控制滑块（0-100）
- 背景音乐音量
- 音效音量
- 语音聊天音量

**画面设置**:
- 画面质量：低/中/高
- 帧率限制：30/60/无限制
- 特效开关：开启/关闭

**控制设置**:
- 自动排列手牌

#### 2.2.4 聊天框面板 ChatBoxPanelUI
**界面布局**:
- 上方区域：聊天记录显示（可滚动）
- 下方区域：输入框和发送按钮

**消息类型**:
- 玩家消息：显示玩家头像、消息内容
- 系统消息：游戏状态变更、玩家进出等

#### 2.2.5 玩家操作面板 PlayerOperationPanelUI
**操作选项**:
- 踢出房间（仅房主可见）
- 设为房主（仅房主可见）
- 添加好友

**权限控制**:
- 房主权限：踢人、开始游戏、修改房间设置
- 普通玩家权限：准备、聊天、查看信息

#### 2.2.6 玩家信息面板 PlayerInfoPanelUI
**显示信息**:
- 玩家头像（大图显示）
- 玩家昵称和ID
- 等级和称号
- 胜率统计
- 历史战绩

#### 2.2.7 提示框面板 PromptPanelUI
**提示类型**:
- 确认对话框（确认/取消）
- 警告提示（单按钮）
- 信息提示（自动消失）
- 加载提示（转圈动画）

## 三、状态管理与数据流

### 3.2 事件处理机制
**面板切换事件**:
- 打开面板：显示对应面板，隐藏其他面板
- 关闭面板：返回大厅主界面
- 面板间通信：通过事件总线传递数据

**玩家状态变更事件**:
- 玩家进入/离开房间
- 玩家准备状态变更

**游戏设置变更事件**:
- 玩法设置更新
- 规则设置更新
- 音量设置更新
