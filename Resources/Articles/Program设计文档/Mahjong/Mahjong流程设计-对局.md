### 1.2 核心功能逻辑

#### 游戏阶段控制
**游戏状态机**:
1. **选庄阶段** (SelectBanker)
   - 四名玩家依次打点，点数最大的为初始庄家
   - 进入发牌阶段

2. **发牌阶段** (DealCards)
   - 按规则发牌（13张/14张）
   - 发牌动画效果
   - 进入出牌阶段

3. **出牌阶段** (PlayCard)
   - 当前玩家出牌
   - 其他玩家可操作（碰、杠、胡）
   - 超时自动出牌

4. **结算阶段** (Settlement)
   - 计算得分
   - 显示结算面板
   - 更新玩家分数

#### 玩家操作逻辑
**出牌流程**:
1. 摸牌：从牌墙摸一张牌
2. 如果能暗杠，可进行杠操作
3. 选择要打出的牌
4. 确认出牌
5. 其他玩家响应操作

**响应操作**:
- **碰牌**：相同牌型立即响应
- **杠牌**：杠
- **胡牌**：满足胡牌条件
- **吃牌**：特定规则下允许
- **过牌**：放弃当前操作机会

#### 音频与特效
**音效系统**:
- 出牌音效
- 碰杠胡音效
- 背景音乐
- 语音聊天

**视觉特效**:
- 出牌动画
- 碰杠胡特效
- 分数变化动画
- 庄家标识动画

interface GameState {
  gamePhase: GamePhase;
  currentPlayer: string;
  players: PlayerGameInfo[];
  deck: Card[];
  discardPile: DiscardInfo[];
  gameSettings: GameSettings;
  roundInfo: RoundInfo;
  operationQueue: Operation[];
}

interface PlayerGameInfo {
  playerId: string;
  handCards: Card[];
  score: number;
  isBanker: boolean;
  isReady: boolean;
  operationTimeLeft: number;
}

interface RoundInfo {
  roundNumber: number;
  bankerPlayer: string;
  wind: WindDirection;

}
